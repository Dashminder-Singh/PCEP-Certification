# PCEP Certification Study Repository

This repository contains my notes, practice scripts, and study materials for the **PCEP – Certified Entry-Level Python Programmer** certification. My goal is to master the fundamentals of Python and track my progress here.

## 📌 Certification Overview
* **Exam Name:** PCEP – Certified Entry-Level Python Programmer (Exam PCEP-30-0x)
* **Passing Score:** 70%
* **Format:** Single-choice, multiple-choice, and fill-in-the-blanks questions

---
